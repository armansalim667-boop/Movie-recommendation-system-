# 🎬 Movie Recommendation System

## 📌 Objective
This project suggests movies to users based on their preferences using Machine Learning techniques. It combines collaborative filtering and content-based filtering to provide personalized recommendations.

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Google Colab
- MovieLens Dataset

## ✨ Features
- Data preprocessing and cleaning
- Collaborative filtering model
- Content-based filtering using movie genres
- User-friendly Streamlit interface
- Fast and personalized suggestions
